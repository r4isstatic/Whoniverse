Whoniverse by r4isstatic

The Whoniverse – a project to make the use of semantic web technologies (e.g. RDF and Ontologies) easier to get into, by means of a ruby/rails application on top of a Triple Store.

The current intention is to load in the complete Ontomedia ontology, which is used to describe the narrative content of media, write an application to present a user with all the available classes/properties, and allow users to begin to construct instances of those classes/properties. The application should also then be able to produce HTML and RDF representations of the defined objects. As of 15th May 2009, about half of the ontology has been loaded into the triple store, and the application can display all the classes and create instances. It also allows new classes to be defined. To be added soon – I’ll list the URLs of each part of the Ontology, together with the appropriate namespace abbreviation.
